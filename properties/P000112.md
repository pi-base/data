The topology contains a coarser topology which is {P53}.

This is equivalent to each of the following:

1. There is a continuous injective map from $X$ into a metrizable space.
2. There is a continuous metric on $X$.

----
#### Meta-properties

- This property is hereditary.
- This property is preserved by arbitrary disjoint unions.
