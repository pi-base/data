Every path component of $X$ is open.

This is equivalent to each of the following:

* $X$ is *weakly locally path connected at $x$* for every $x\in X$;
that is, each $x$ has an open path connected neighborhood.

* $X$ is *weakly locally path connected im kleinen at $x$* for every $x\in X$;
that is, each $x$ has a path connected neighborhood.

This property is called *PCGS* in {{zb:0451.54017}}.

Compare with {P42}.

----
#### Meta-properties

- $X$ satisfies this property iff its Kolmogorov quotient $\text{Kol}(X)$ does.
- This property is hereditary with respect to clopen sets.
- This property is preserved by finite products.
- This property is preserved by box products.
- This property is preserved by arbitrary disjoint unions.
