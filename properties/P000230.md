$X$ has a base consisting of {P199} sets.

Note that there is another usage of the term "locally contractible" ({P229}). We are following the naming conventions from {{mathse: 5109428}}.

----
#### Meta-properties

- This property is hereditary with respect to open sets.
- $X$ satisfies this property if its Kolmogorov quotient $\text{Kol}(X)$ does.
- This property is preserved by arbitrary disjoint unions.
