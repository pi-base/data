Every continuous map from the space to itself has at least one fixed point.

----
#### Meta-properties

- This property is hereditary with respect to retracts.
A set $A\subseteq X$ is called a [*retract*](https://en.wikipedia.org/wiki/Retraction_(topology)) if there exists a continuous $f:X\to A$ fixing each point of $A$.
