Let $Y$ denote the subspace of successor cardinals in $X$, that is,
$Y=\{\beta+1:\beta\in\omega_1\}$. Note that $|Y|=|X|$.

Because $\omega_1$ is an initial ordinal, $(Y,\le)$ is order-isomorphic to $(\omega_1,\le)$.
And since an element $\alpha=\beta+1\in Y$ has $[\alpha,\to)=(\beta,\to)$ as smallest neighborhood in $X$,
it has $[\alpha,\to)\cap Y$ as smallest neighborhood in $Y$.
This shows that the subspace topology on $Y$ is homeomorphic to {S220}.

But {S220} is not homeomorphic to {S221}, for example because only the first one is {P90}.
