Let $X$ be the set of positive integers. For $S \subset X$, let $N(n, S) = |\{m \in S\ |\ m < n\}|$. Declare $U \subset X$ to be open if $1 \not\in U$, or if $1 \in U$ and $\lim_{n \rightarrow \infty} N(n, U)/n = 1$.
