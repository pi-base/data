Extend $\varphi$ to $X$ so that if $E = E_r = \{\varphi^{-1}(s_n) : n\in\omega\}$ and $s_n\to r$ then $\varphi(p_E) = r$. If $\varphi(p_E)\in U$ where $U\subseteq \mathbb{R}$ is open, find $N$ such that $s_n\in U$ for $n\geq N$, then $V =\overline{E}\setminus\varphi^{-1}(\{s_1, s_2, ..., s_N\})$ is an open neighbourhood of $p_E$ and $\varphi(V)\subseteq U$. So $\varphi:X\to\mathbb{R}$ is a continuous injection, hence $X$ is submetrizable.
