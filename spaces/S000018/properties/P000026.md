For any $D = \{(x_n, a_n)\ |\ n \in \omega\} \subset X \times \{0,1\}$, $U = \{(y,z)\ |\ y \neq x_n \text{ for any } n\}$ is a basic open set disjoint from $D$ and so $D$ is not dense.
