Let $\mathcal{A}$ be an almost disjoint family (or ADF) of size $\mathfrak{c}$ on $\mathbb{N}$ (see chapter 12 of {{zb:1471.54001}} for a definition of ADF, and why ADF of size $\mathfrak{c}$ exists). Every element of $\mathcal{A}$ is a countably infinite set, and so for every $S\in\mathcal{A}$ there exists $p_S\in \overline{S}\setminus S$. Let $Y = \{p_S : S \in \mathcal{A}\}$. If $S\in\mathcal{A}$ then $\overline{S}\subseteq X$ is open and $\overline{S}\cap Y = \{p_S\}$ since $(\overline{S}\setminus S)\cap(\overline{S'}\setminus S') = \emptyset$ for distinct $S, S'\in \mathcal{A}$ (here note that necessarily $\overline{S}\setminus S\subseteq \beta\omega\setminus \omega$ for $S\in\mathcal{A}$). So $Y\subseteq X$ is a discrete set of size continuum.
