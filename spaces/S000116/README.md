For each $n \in \omega$ let $L_n$ be the closed line segment from $(0,0)$ to $(1,\frac{1}{n+1})$.
The Infinite Broom is the set $X=\bigcup_{n \in \omega} L_n \cup \big((\frac{1}{2},1] \times \{0\}\big) \subset \mathbb{R}^2$ with the subspace topology.

This space is a subspace of {S117}.
