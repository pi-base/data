Let $X = \{0,1,2\}$ with open right rays $\{\{0,1,2\}, \{1,2\}, \{2\}, \emptyset\}$.
(Equivalently could use the left rays $\{\emptyset, \{0\}, \{0,1\}, \{0,1,2\}\}$ instead.)

Compare with {S42}, counterexample #50 ("Right Order Topology on $\mathbb R$")
in {{zb:0386.54001}}.
