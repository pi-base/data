Let $x \in X$.
If $x = \infty$, all neighborhoods of $x$ are cofinite, so we can follow the same argument as in {T800}.
Otherwise $x$ is isolated, so there cannot be a countably infinite set converging to $x$. Hence, the assertion {P210} is trivially true.
