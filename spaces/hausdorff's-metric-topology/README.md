name: Hausdorff's Metric Topology
---
For any metric space $(X,d)$, Hausdorff's Metric Topology is the set of all closed and bounded subsets of $X$ with the metric $d^*(A,B) = \max\{ \sup_{a \in A}\ d(a,B), \sup_{b \in B}\ d(b,A) \}$
