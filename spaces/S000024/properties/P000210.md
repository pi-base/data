If $x \in \{\infty_1,\infty_2\}$, all nonempty neighborhoods of $x$ are cofinite, so we can follow the same argument as in {T800}.

Otherwise $x$ is isolated, so there cannot be a countably infinite set converging to $x$, so the statement becomes trivial.
