The closures of any two base elements $U_a(b)$ and $U_c(d)$ intersect in infinitely many points.

However if there were a cut point $p \in X$, there would have to be base elements $U_{a'}(b'), U_{c'}(d')$, such that $\overline{U_{a'}(b')} \cap \overline{U_{c'}(d')} \subseteq \{p\}$.
