The open cover $\{S_n\}$ has no proper refinement but $S_1 = (0,1) \cup (1,2)$ intersects every set in the cover, so $\{S_n\}$ is not locally finite.

See item #6 for space #54 in {{doi:10.1007/978-1-4612-6290-9_6}}.
