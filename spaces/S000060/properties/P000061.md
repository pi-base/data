  name: "Prove that countable intersection/union of zero/cozero sets is a zero/cozero set."
---

Note that for $q \in \mathbb{Q}$, $f : X \to \mahtbb{R}, x \maps $\chi_{\{q\}}(x)$ is continuous. Therefore any singleton in $\mathbb{Q}\subseteq X$ is a cozero set and therefore any subset $A \subseteq \mathbb{Q}$ as well, see {{mathse:4465592}}.

So given a cozero set $U\subseteq X$, $V := \mathbb{Q}\setminus U$ is a cozero set and disjoint from $U$. Then $\mathbb{Q}\subseteq U \cup V$, but $\mathbb{Q}$ (and therefore $U \cup V$) is dense in $X$ already.
