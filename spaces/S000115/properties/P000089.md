Define $f : X \to X$ as
$$f((x, y)) =\begin{cases}
(\frac{2}{\pi},\, 1) 
    & \text{if } x \leq \frac{2}{5\pi} \\
(-x + \frac{12}{5\pi},\, \sin(1 / (-x + \frac{12}{5\pi})))
    & \text{if } \frac{2}{5\pi} \leq x \leq \frac{2}{\pi},\ y = 1 \\
(-x + \frac{12}{5\pi},\, 1)
    & \text{if } \frac{2}{5\pi} \leq x \leq \frac{2}{\pi},\ y = \sin(\frac{1}{x})\\
(\frac{2}{5\pi},1) & \text{if } \frac{2}{\pi} \leq x \\
\end{cases}.$$
Then $f$ has no fixed point and is continuous.
