Let $X = \{x_n: n \in \mathbb{N}\}$ be dense and, for each $p \in X$, $\{V_n(p): n \in \mathbb{N}\}$ be a countable local $\pi$-base for $p$. Then $\{V_n(x_m): n, m \in \mathbb{N}\}$ is a countable (global) $\pi$-base.

To see this, if $O$ is open, then there is some $m$ so that $x_m \in O$ by hypothesis, and so there is $n$ where $V_n(x_m) \subseteq O$, as desired.

This is a special case of Theorem 3.8(b) of {{zb:0559.54003}}.
