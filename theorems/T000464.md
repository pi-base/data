If $x\in U$ where $U$ is open, then there exists a zero-set $Z$ and a cozero set $W$ such that $x\in Z\subseteq W\subseteq U$. Since $X$ is {P217}, we can find a clopen set $V$ such that $x\in Z\subseteq V\subseteq W\subseteq U$. So $X$ is {P50}.
